eCorcoran

The Intranet for real estate agents to get the resources they need to close their deals.

  My Experience

Corcoran is a real estate firm and is redesigning its intranet (ecorcoran.com). I am working on the redesign of ecorcoran.com on a team of five to improve agents' user experience.

(1) Participated in user interviews to identify agents' complaints and problems.

(2) Designed and built the user interface of ecorcoran.com.

(3) Prototyped templates for different sections based on the needs of agents in different regions.

(4) Collaborated closely with the back-end developer to implement data into the front-end and improve code quality.

  User Interview & Problem Identification

We interviewed five agents that provided broad insights into their current workflow and pain points. We identified existing problems and generated a user journey map.

Problem #1: Searchability is weak because document retrieval is cumbersome.

Problem #2: Lacking easy access to top or frequently used resources.

Problem #3: Unfriendly mobile experience; hard to use on the go.

User Journey Map (Take renting as an example.)

  Solutions

#1 Powerful Search Engine:

  • Access to search all over the site.
  • Filtering and sorting enables agents to drill down into subsets of search results.
  • When searching by keywords, an agent could refine his search criteria by category.

#2 Personalized Content:

  • Homepage cotent was populated based on region and user type.
  • Links and documents were ordered by views and date to bring out the most recent and frequently used resources to the top.

#3 Responsive Design:

  • Optimized for any mobile device.

Responsive Design (mobile and tablet)

  Iteration

After we released the eCorcoran version 1 for one and a half monthes, we conducted usability test with agents and collected their feedback.

The agents' main complaints are:

#1 Too much scrolling on all pages.

#2 The organization of the search results is difficult to understand.

#3 AgentSuite section is overwhelming and confusing.

Usability Test Summary (Feb. 2017)

Based on agents' feedback, we made the following enhancements:

#1 A more condensed layout, which shows more information on one screen.

#2 Customized filters for different sections. Xore descriptive filters and sorting.

#3 Allow agents to manage options in AgentSuite section by themselves.
