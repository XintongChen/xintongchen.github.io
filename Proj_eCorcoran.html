  My Experience

Corcoran is a real estate firm and is redesigning its intranet (ecorcoran.com). I am working on the redesign of ecorcoran.com on a team of four to improve agents' user experience.

(1) Participated in user interviews to identify agents' complaints and problems.

(2) Designed and built the user interface of ecorcoran.com.

(3) Prototyped templates for different sections based on the needs of agents in different regions.

(4) Collaborated closely with the back-end developer to implement data into the front-end and improve code quality.

  User Interview & Problem Identification

We interviewed five agents that provided broad insights into their current workflow and pain points. We identified existing problems and generated a user journey map.

Problem #1: Searchability is weak because document retrieval is cumbersome.

Problem #2: Lacking easy access to top or frequently used resources.

Problem #3: Unfriendly mobile experience; hard to use on the go.

User Journey Map (Take renting as an example.)

  Solutions

#1 Powerful Search Engine: When searching by keywords, an agent could refine his search criteria by category.

#2.1 Filtered Resources: Links and documents were ordered by views and date to bring out the most recent and frequently used resources to the top.

#2.2 Personalized Content: Homepage cotent was populated based on region and user type.

#3 Responsive Design: Optimized for any mobile device.
